Swift Integration

Swift Integration

Clicktale for Apps is compatible with Apple’s new programming language Swift. Follow these steps for creating a new project

  • Create a project with Xcode
  • Select “Swift” as programming language
  • Create a file ( Header file .h ) and add it to your project( for example : YourProjectName-Bridging-Header.h )
  • Copy and paste the code below to YourProjectName-Bridging-Header.h
#import <Clicktale/Clicktale.h>
  • Modify Xcode project “Build Settings -> Objective-C Bridging Header” and set it to “YourProjectName\YourProjectName-Bridging-Header.h” as a path.

  • Add the SDK to your your project. It is the same SDK for Objective-C and Swift.

  • Initialize Clicktale for Apps in AppDelegate.swift


import Clicktale


func application(application: UIApplication, didFinishLaunchingWithOptions launchOptions: [NSObject: AnyObject]?) -> Bool { 

    // Override point for customization after application launch. 
    Clicktale.sharedInstance().setAccessKey("YOUR_ACCESS_KEY", secretKey: "YOUR_SECRET_KEY") 
    Clicktale.sharedInstance().start() 
    return true 
}
