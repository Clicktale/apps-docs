Privacy

Privacy

As our SDK will record everything displayed on your clients screens, it is important to have the ability to hide sensitive information or presonally identifyable information (PII).

The way it will be seen in the rendered video will be a black box around wherever was set to be hidden.

Hide any UIView or UITextField

In our iOS SDK, you will need to use two methods in order to hide any UIView.

One for starting the privacy mode for views, and one for stopping it.

It is important to stop it so there will be no performance hit on the application later on.

-(void)startPrivacyForViews:(NSArray <UIView *> *)views; 
-(void)stopPrivacyForAllViews;

Or with Swift

func startPrivacy(forViews controls: [Any]!)
func stopPrivacyForAllViews()

You should try to start privacy in - (void)viewWillAppear:(BOOL)animated and stop it in (void)viewDidDisappear:(BOOL)animated

Example:

-(void)viewWillAppear:(BOOL)animated
{ 
    [[Clicktale sharedInstance] startPrivacyForViews: @[self.viewSensitive, self.viewPrivate, self.textFieldSensitive]]; 
    [super viewWillAppear:animated];
} 

-(void)viewDidDisappear:(BOOL)animated
{ 
    [super viewDidDisappear:animated];
    [[Clicktale sharedInstance] stopPrivacyForAllViews]; 
}

Or with Swift:

override func viewWillAppear(animated: Bool) {
    Clicktale.sharedInstance().startPrivacy(forViews: [self.viewPrivate, self.viewPrivate, self.textFieldSensitive]) 
}

override func viewDidDisappear(animated: Bool) { 
    super.viewDidDisappear(animated)
    Clicktale.sharedInstance().stopPrivacyForAllViews() 
}

Secure UITextField

The SDK automatically hides any secure UITextField, the automatic hiding happens only once the user focus into the field on the first time.

Hide the whole screen

It is possible to hide the whole screen, by that manner the recording shal show a black scrren until you stop hidding it:

  .......
  //In any point that you do not want to capture the screen
  [[Clicktale sharedInstance] startHidingScreen];
  .....

Or with Swift:

    Clicktale.sharedInstance().startHidingScreen()

Do not forget to stop hidding

  .......
  //In any point that you want to resume capturing the screen
  [[Clicktale sharedInstance] stopHidingScreen];
  .....

Or with Swift:

    Clicktale.sharedInstance().stopHidingScreen()

In addtion you can check whether the screnn is hidden 

  .......
  [[Clicktale sharedInstance] isScreenHidden];
  .....

Or with Swift:

    Clicktale.sharedInstance().isScreenHidden()

PII in WebViews

In a case you have WebViews in your application (UIWebView or WKWebView) that has web-pages you control and have some PII on them you want to hide.

  1. Use the 'ctHidden' CSS class on that HTML element
     <input id="pass1" type="text" name="password" class="ctHidden">

  2. Call the following API to start privacy on the view (UIWebView or WKWebView)

    -(void)viewWillAppear:(BOOL)animated
{
 NSArray *sensitiveViewsReferenceArray = @[self.webView];
 // Then send that array to SDK
 [[Clicktale sharedInstance] startPrivacyForWebViews:sensitiveViewsReferenceArray];

 [super viewWillAppear:animated];
}

    Or with Swift:

    override func viewWillAppear(animated: Bool) {
 Clicktale.sharedInstance().startPrivacy(forWebViews: [self.webView]) 
}
  3. Call the following API to stop privacy on the view
    -(void)viewDidDisappear:(BOOL)animated
{
[super viewDidDisappear:animated]; 
[[Clicktale sharedInstance] stopPrivacyForAllWebViews];
}
    Or with Swift:
    override func viewDidDisappear(animated: Bool) { 
 super.viewDidDisappear(animated)
 Clicktale.sharedInstance().stopPrivacyForAllWebViews() 
}

More Granularity

In some cases, you might want to have more control over hiding or showing different elements of your application during the playback, at any time during the view controller lifetime.

This can be done via using the startPrivacy API as above, and in order to stop privacy for a specific view at a specific time during the lifetime of your application or view controller.

  [[Clicktale sharedInstance] stopPrivacyForViews: @[self.viewSensitive]];
  [[Clicktale sharedInstance] stopPrivacyForWebViews: @[self.webView]];

Or with Swift:

  Clicktale.sharedInstance().stopPrivacy(forViews: [self.viewPrivate]) 
  Clicktale.sharedInstance().stopPrivacy(forWebViews: [self.webView])

Stop and clear all PII

You can clear all the PII list on the SDK with this method:

  [[Clicktale sharedInstance] stopPrivacyForAll];

Or with Swift:

  Clicktale.sharedInstance().stopPrivacyForAll()

UITableViewController & PII

When working with UITableViewController and PII you should specify for each component in the cell that you would like to hide and because of cell reuse you sould remove this component at reuse:

For example, this is a custom cell:

@interface CTTableViewCell : UITableViewCell
@property (weak, nonatomic) IBOutlet UITextField *sTextField;
@property (weak, nonatomic) IBOutlet UIView *sView;
@end

Manage cell Hide/Show components

- (UITableViewCell *)tableView:(UITableView *)tableView cellForRowAtIndexPath:(NSIndexPath *)indexPath {

    CTTableViewCell *cell = [tableView dequeueReusableCellWithIdentifier:@"Cell" forIndexPath:indexPath];

    if(indexPath.row % 2 == 0)//Decide if we want to hide components
    {
        cell.sView.backgroundColor = [UIColor redColor];
        cell.sTextField.text = @"DO NOT SHOW ME!!!";
        [[Clicktale sharedInstance] startPrivacyForViews:@[cell.sView, cell.sTextField]];        
    }
    else
    {
        cell.sView.backgroundColor = [UIColor yellowColor];
        cell.sTextField.text = @"HI";
        [[Clicktale sharedInstance] stopPrivacyForViews:@[cell.sView, cell.sTextField]];
    }

    return cell;
}

UICollectionViewController & PII

When working with UICollectionViewController and PII you should specify each UICollectionViewCell that you would like to hide and because of cell reuse you should remove this cell from PII at reuse:

Manage your hidden cells:

var hiddenCells = [UICollectionViewCell]()

Manage cell Hide/Show components

 override func collectionView(_ collectionView: UICollectionView, cellForItemAt indexPath: IndexPath) -> UICollectionViewCell {

    let cell = collectionView.dequeueReusableCell(withReuseIdentifier: "Cell", for: indexPath) as! ItemCell

    //Select cell to Hide PII                  
    if(indexPath.row % 2 == 0)
    {
        hiddenCells.append(cell)
        Clicktale.sharedInstance().startPrivacy(for: [cell])
    }
    else
    {
        if let index = hiddenCells.index(of: cell)
        {
               hiddenCells.remove(at: index)
               Clicktale.sharedInstance().stopPrivacy(for: [cell])
        }
    }

   return cell
}

 override func viewDidDisappear(_ animated: Bool) {
    super.viewDidDisappear(animated)
    Clicktale.sharedInstance().stopPrivacy(for: hiddenCells)
}

override func viewWillAppear(_ animated: Bool) {
    super.viewWillAppear(animated)
    Clicktale.sharedInstance().startPrivacy(for: hiddenCells)
}

Pause Screen Recording

There is an ability to pause and resume acreen recording

Pause Screen Recording:

 [[Clicktale sharedInstance] pauseScreenRecording];

Or with Swift:

 Clicktale.sharedInstance().pauseScreenRecording()

Resume Screen Recording:

 [[Clicktale sharedInstance] resumeScreenRecording];

Or with Swift:

 Clicktale.sharedInstance().resumeScreenRecording()

Check if the screen is Recorded:

 [[Clicktale sharedInstance] isScreenRecorded];

Or with Swift:

 Clicktale.sharedInstance().isScreenRecorded()
