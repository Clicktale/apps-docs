Getting Started

Getting Started with iOS

Adding the Framework to your project

The SDK is available via CocoaPods. If your project supports CocoaPods, you can add the following line to your Pod file:

pod ...
pod 'Clicktale'
pod ...

The Clicktale pod will add the framework.

Manual Integration

In order to add the framework manually, follow the next steps:

  1. You will need obtain the Clicktale.framework

    You can find it at: https://github.com/Clicktale/ios-sdk/raw/master/Clicktale.zip

  1. Add the Clicktale.framework to your project

  2. Open your project settings > General > Embedded Binaries and add the Clicktale.framework

  3. Import Clicktale.h file to access public methods of the framework.

    #import <Clicktale/Clicktale.h>

  4. If working with Swift, please read the Swift integration guide.

Authentication

In order to ensure that only you can send data to your project, you are required to send your access key and secret key to authenticate with our server before you begin recording.

At the end of your - application:didFinishLaunchingWithOptions: function, just before returning, add the following code:

- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {
    //your code 

    [[Clicktale sharedInstance] setAccessKey:@"YOUR_ACCESS_KEY" secretKey:@"YOUR_SECRET_KEY"]; 
    [[Clicktale sharedInstance] startClicktale]; 

    return YES;
}

Or with Swift:

func application(application: UIApplication, didFinishLaunchingWithOptions launchOptions: [NSObject: AnyObject]?) -> Bool { 
    // Override point for customization after application launch. 

    Clicktale.sharedInstance().setAccessKey("YOUR_ACCESS_KEY, secretKey: "YOUR_SECRET_KEY") 
    Clicktale.sharedInstance().start() 
    return true
}

Start Recording

After you are authenticated, you will need to call the -startClicktale: method to begin the actual recording.

By default, the recording is stopped and uploaded to Clicktale server when session ends and the app is minimized to the background.

[[Clicktale sharedInstance] startClicktale];

Clicktale.sharedInstance().start()

This method creates a new session and starts recording the session. All the sensor & device information starts to be uploaded after startClicktale is called.

If you don’t use “Storyboards” in your Xcode project, you must call Clicktale methods after UIWindow created.

... 
[window addSubView:myViewController]; 
[[Clicktale sharedInstance] startClicktale]; 
...
