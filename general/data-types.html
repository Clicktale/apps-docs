Data Types

Data Types

Clicktale for apps collects two main data types:

  • Session Playbacks
  • Meta-data

Session Playbacks

Clicktale records your user activity within the app and allows you to playback real user sessions.

The recorded session are accessible for three months (90 days). After this period the playback videos will be deleted.

Metadata

Clicktale for apps collects metadata which includes all other analytical functionality such as events, screen views, gestures and more.

Metadata is saved for over a year – this data is used to generate the different reports and deeper insights that you see inside the Clicktale for Apps platform.

The data retention value can be changed by request per application – please contact Clicktale to find out more information about changing the data retention duration.
