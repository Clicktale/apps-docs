Hybrid Applications

Hybrid Applications Support

Clicktale for Apps SDK supports recording and providing analytics for Hybrid Applications.

A hybrid application can either be based on a framework such as Cordova or an application which displays data from a WebView like object.

Because of the way our SDK captures data - the resulting session video will automatically capture and include data coming from hybrid content.

For more infomation about Cordova, please see the Cordova Integration instructions.

When it comes to hiding sensitive information or passing custom events from within the content coming from the web view, some customization may be required, please look at the description at the different Privacy pages for more information.
