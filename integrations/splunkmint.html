Splunk Mint

Splunk Mint

When using both Clicktale for Apps and Splunk Mint, you can now integrate the systems and be able to see inisde Splunk Mint, an event leading to the corresponding session inside Clicktale.

Splunk Mint has three data point which can be used for this integration

  • Events
  • Breadcrumbs
  • ExtraData

In Clicktale for Apps, when a session begins, which may be on first activation (‘splash screen’ like) or when resuming from background after some time of inactivity, a delegate is used to signal that a new session has begun. At that point you should be sending Splunk Mint an event with the URL for that session recording. In addition you can send the session link as a crash report breadcrumb or as "ExtraData".

Sending the Clicktale session link to Splunk

The delegate function should be used like this for iOS:

-(void)clicktaleDidSessionURLCreated:(NSString *)sessionLink
{ 
  //Send a Clicktale Session url event to Splunk Mint 
  MintLimitedExtraData *extraData = [[MintLimitedExtraData alloc] init];
  [extraData setValue:sessionLink forKey:@"SessionURL"];
  [[Mint sharedInstance] logEventWithName:@"Clicktale" logLevel:InfoLogLevel extraData:extraData];

  //Send Extra Data      
  [[Mint sharedInstance] addExtraData:sessionLink forKey:@"ClicktaleSessionURL"];

  //Add breadcrumb      
  [[Mint sharedInstance] leaveBreadcrumb:[NSString stringWithFormat:@"ClicktaleSessionURL: %@",sessionLink]];
}

Or with Swift:

func clicktaleDidSessionURLCreated(_ sessionLink: String!) 
{  
 //Send a Clicktale Session url event to Splunk Mint 
  let extraData = MintLimitedExtraData()
  extraData.setValue(sessionLink, forKey: "SessionURL")
  Mint.sharedInstance().logEvent(withName: "Clicktale", logLevel: InfoLogLevel, extraData: extraData)

  //Send Extra Data      
  Mint.sharedInstance().addExtraData(sessionLink, forKey: "ClicktaleSessionURL")

  //Add breadcrumb      
  Mint.sharedInstance().leaveBreadcrumb("ClicktaleSessionURL: " + sessionLink)
}

Or in Android:

@Override
public void onSessionURLCreated(String sessionLink) { 
  //Send a Clicktale Session url event to Splunk Mint 
  Mint.logEvent("Clicktale", MintLogLevel.Info, "SessionLink", sessionLink);

  //Send Extra Data      
  Mint.addExtraData("ClicktaleSessionLink", sessionLink);

  //Add breadcrumb      
  Mint.leaveBreadcrumb("ClicktaleSessionLink: " + sessionLink);
}

The above is just one suggestion on how to send the sessions link, it is also possible to send the event at your convenience at any other time during the runtime by calling:

// other code

//Send a Clicktale Session url event to Splunk Mint 
MintLimitedExtraData *extraData = [[MintLimitedExtraData alloc] init];
[extraData setValue:[[Clicktale sharedInstance] getSessionLink] forKey:@"SessionURL"];
[[Mint sharedInstance] logEventWithName:@"Clicktale" logLevel:InfoLogLevel extraData:extraData];

//Send Extra Data      
[[Mint sharedInstance] addExtraData:[[Clicktale sharedInstance] getSessionLink] forKey:@"ClicktaleSessionURL"];

//Add breadcrumb      
[[Mint sharedInstance] leaveBreadcrumb:[NSString stringWithFormat:@"ClicktaleSessionURL: %@",[[Clicktale sharedInstance] getSessionLink]]];

// other code

Or with Swift:

// other code

//Send a Clicktale Session url event to Splunk Mint 
 let extraData = MintLimitedExtraData()
 extraData.setValue(Clicktale.sharedInstance().getSessionLink(), forKey: "SessionURL")
 Mint.sharedInstance().logEvent(withName: "Clicktale", logLevel: InfoLogLevel, extraData: extraData)

//Send Extra Data      
 Mint.sharedInstance().addExtraData(Clicktale.sharedInstance().getSessionLink(), forKey: "ClicktaleSessionURL")

//Add breadcrumb      
 Mint.sharedInstance().leaveBreadcrumb("ClicktaleSessionURL: " + Clicktale.sharedInstance().getSessionLink())

// other code

Android:

// other code

//Send a Clicktale Session url event to Splunk Mint 
 Mint.logEvent("Clicktale", MintLogLevel.Info, "SessionLink", Clicktale.getSessionLink());

//Send Extra Data      
Mint.addExtraData("ClicktaleSessionLink", Clicktale.getSessionLink());

//Add breadcrumb
Mint.leaveBreadcrumb("ClicktaleSessionLink: " + Clicktale.getSessionLink());

// other code
  • Notice that the Clicktale.getSessionLink() method will return an empty string if the session link is not ready yet.
