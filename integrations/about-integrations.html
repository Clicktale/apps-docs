About Data Synchronization

About Data Synchronization

Given the plethora of available analytic platforms and various advantages and disadvantages of each, Clicktale understands that no single analytic platform or SDK covers all of the information you are looking for.

We create and maintain data integrations with other vendors to help you uncover deep insights and understand the context of the data that you are getting from your existing analytic and marketing ecosystem. This includes all of your favorite optimization, VoC and analytics solutions. Bridge the data sets from every system to gain a comprehensive picture of your enterprise’s digital customer experience.

The basic data integration with Clicktale for Apps starts by linking between the Clicktale session or user into the partner system – usually via sending a link as meta-data into the third party platform or attaching an event data record inside of Clicktale, for easy data access from the partner system.

iOS Delegates

Inside your AppDelegate.h file you should add to the UIResponder the ClicktaleDelegate interface

@interface AppDelegate : UIResponder <UIApplicationDelegate,ClicktaleDelegate>

Then, inside AppDelegate.m you will need to implement the two delegates as follows

-(void)clicktaleDidSessionURLCreated:(NSString *)sessionLink
{
    NSLog(@"Current Session URL : %@",sessionLink);
}

-(void)clicktaleDidSessionIDCreated:(NSString *)currentSessionID
{
    NSLog(@"Current Session ID : %@",currentSessionID);
}

And inside your didFinishLaunchingWithOptions function, add the setDelegate to self

    [[Clicktale sharedInstance] setDelegate:self];

It is important to set the delegate before calling the -startFlight function. This is because the startFlight function manages the creation of IDs and sessions, and a session ID may be changes when the app is staying for a long time in the background.

Android Callbacks

In android, we supply two callbacks onSessionURLCreated and onSessionIDCreated. They act similarly to the iOS equivalents, and when calling Clicktale.start() function you should pass the class which implements these callbacks.

And example can be seen in Getting Started article of the android documentation.
